Проектная работа — интерактивная веб-викторина на HTML, CSS и JavaScript.

Структура проекта

```
quiz-project/
├── index.html       — разметка приложения
├── css/
│   └── style.css    — стили (прогресс-бар, индикация правильного/неправильного ответа)
├── js/
│   └── main.js      — логика викторины (вопросы, подсчёт баллов, переходы)
└── README.md
```

## Как запустить

Откройте файл `index.html` в любом современном браузере — установка не требуется.

Функции:
- 10 вопросов с 4 вариантами ответа каждый
- Прогресс-бар, обновляющийся с каждым вопросом
- Подсчёт баллов в реальном времени
- Правильный ответ выделяется **зелёным**, неправильный — **красным**
- Экран результата с оценкой и кнопкой «Начать заново»
